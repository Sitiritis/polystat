To build it you need to have LaTeX, GNU aspell,
[texqc](https://github.com/yegor256/texqc),
and
[texsc](https://github.com/yegor256/texsc) installed. 
Then, just do this:

```bash
$ latexmk -pdf
```

Everything will be built just fine and you will get the `.pdf` file.
